# Containerized React/Express/Mongodb sample app

> A basic step to containerize an React/Express/Mongodb app in Docker

## Quick Start

1. Clone this repo

```
# Run
docker-compose up

# Stop
docker-compose down

# To re-build
docker-compose build

# Auto Run after building
docker-compose run --build
```
